.. _cream:

Cream Utility
=============

Cream is a utility programme for SCONE. By this we mean that it contains number
of scripts that automate common operations and make set-up of SCONE environment
easier.

It is intended to be used as an executable, however it is possible to import
all of its modules directly into other Python scripts.
